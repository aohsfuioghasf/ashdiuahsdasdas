# Upload These Files To GitHub

Upload every file in this folder to the root of your GitHub repo.

Do not upload the folder itself. Open this folder, select all files, then drag
them into GitHub.

Render settings:

```text
Root Directory: leave blank
Build Command: npm install
Start Command: npm start
```

Render environment variables:

```ini
NODE_ENV=production
STARBLAST_REGION=Europe
STARBLAST_ECP_KEY=your-ecp-key
DASHBOARD_USERS=Omega:your-password,Pasha:your-password
SESSION_TTL_HOURS=12
```
